# Registre des incidents de confidentialité

> **Obligation légale :** Loi 25 du Québec, art. 3.5 — Tout incident de confidentialité impliquant des renseignements personnels doit être consigné dans un registre conservé pendant au moins 5 ans.
>
> **Responsable :** Ralph Christian Gabriel, Fondateur — privacy@dpmcalendar.com

---

## Format d'entrée

Pour chaque incident, documenter les éléments suivants :

| Champ | Description |
|-------|-------------|
| **Numéro** | INC-YYYY-NNN (ex: INC-2026-001) |
| **Date de découverte** | YYYY-MM-DD HH:MM |
| **Date de l'incident** | YYYY-MM-DD (si différente de la découverte) |
| **Description** | Nature de l'incident (accès non autorisé, perte, vol, divulgation, etc.) |
| **Renseignements concernés** | Types de données touchées (noms, emails, données de calendrier, etc.) |
| **Nombre de personnes touchées** | Nombre estimé ou exact |
| **Cause** | Cause technique ou humaine identifiée |
| **Évaluation du risque** | Risque de préjudice sérieux? (OUI/NON) — Critères : sensibilité des données, conséquences anticipées, probabilité d'utilisation malveillante |
| **Mesures correctives** | Actions prises pour contenir et résoudre l'incident |
| **Notification CAI** | Notifié? (OUI/NON) — Date de notification |
| **Notification personnes** | Personnes notifiées? (OUI/NON) — Date et méthode |
| **Statut** | OUVERT / EN COURS / FERMÉ |
| **Date de clôture** | YYYY-MM-DD |
| **Leçons apprises** | Améliorations à implémenter pour prévenir la récurrence |

---

## Critères d'évaluation du risque de préjudice sérieux

(Loi 25, art. 3.5 — L'évaluation doit tenir compte de :)

1. **Sensibilité des renseignements** — Données de santé (energy logs, mood), données financières, identifiants, données de localisation
2. **Conséquences anticipées** — Vol d'identité, atteinte à la réputation, préjudice financier, détresse psychologique
3. **Probabilité d'utilisation malveillante** — Données chiffrées vs en clair, accessibilité des données, nature de l'attaquant

**Si le risque de préjudice sérieux est évalué comme OUI :**
- Notifier la Commission d'accès à l'information (CAI) dans les meilleurs délais
- Notifier les personnes touchées dans les meilleurs délais
- Documenter les notifications dans ce registre
